# Conta Aberta

App web independente, mobile-first, para controle de fiados em bares.

## Funções desta primeira versão

- Dashboard com total a receber, vencidos e recebidos hoje
- Cadastro simples de clientes
- Lançamento de fiado
- Recebimento de pagamento total ou parcial
- Histórico por cliente
- Status automático: liberado, atenção e bloqueado
- Cobrança por WhatsApp com mensagem pronta
- Dados salvos no navegador via `localStorage`

## Como rodar

```bash
npm install
npm run dev
```

Depois abra o endereço mostrado no terminal.

## Próximo passo recomendado

Depois de testar o fluxo no bar, conectar ao Supabase com tabelas reais, login e permissões.
